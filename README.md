# CodSoft_Task5
# Task 5: Credit Card Fraud Detection

## About the Project

This project was completed as part of the **CodSoft Data Science Virtual Internship**. The objective is to build a machine learning model that accurately identifies fraudulent credit card transactions based on historical transaction data. The project demonstrates data preprocessing, exploratory data analysis, binary classification model development, and performance evaluation for fraud detection.

## Technologies Used

* Python
* Visual Studio Code (VS Code)
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Project Workflow

* Data Collection
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Fraud Transaction Analysis
* Model Training
* Model Evaluation
* Fraud Detection Prediction

## Outcome

This project strengthened my understanding of binary classification, fraud detection techniques, data preprocessing, and machine learning model evaluation using real-world financial transaction data.
